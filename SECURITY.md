# Security Policy

## Supported Versions

Currently only the following versions of the Ghost MCP are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us by:

1.  Opening a GitHub Security Advisory in this repository (preferred).
2.  Alternatively, if you cannot use the security advisory feature, please contact the maintainer directly through the project's official channels.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact and any known mitigations.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please do not report security vulnerabilities through public GitHub issues.
